## 运行环境 ##
Eclipse版本：Oxygen.2 Release (4.7.2)<br/>
JDK版本：jdk1.8.0_115

## 所用到的技术栈 ##
### 前端 ###
HTML5<br/>
thymeleaf<br/>
EasyUI<br/>

### 后端 ###
Spring Boot: 2.0.5<br/>
Spring Boot-Data-Jpa: 2.0.5<br/>
commons-codec: 1.11<br/>
lombok: 1.16.22<br/>
拦截器<br/>

### 数据库 ###
MySQL
